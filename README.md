# BlogSite Project
## How To Run Locally
1- Install dependencies
```bash
pnpm run install
```
2- Follow these steps to start the backend service
```bash
# Navigate to backend directory
cd backend

# Install dependencies
pnpm install

# Generate Prisma
npx prisma generate

# Start dev server
pnpm run start:dev
```

3- Follow these steps to start the frontend service
```bash
# Navigate to backend directory
cd frontend

# Install dependencies
pnpm install

#Start the service
pnpm run dev
```
